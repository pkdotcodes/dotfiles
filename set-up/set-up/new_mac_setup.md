# New MacBook Setup

## Install Homebrew

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

## Keyboard Settings
- **Caps Lock → Escape**
  *System Settings → Keyboard → Keyboard Shortcuts → Modifier Keys*

- **Fast Key Repeat, Short Delay**
  *System Settings → Keyboard*
  - Key Repeat: **Fast**
  - Delay Until Repeat: **Short**

- **Add German Input**
  *System Settings → Keyboard → Input Sources → Add Input Source*

## Trackpad
- **Enable Tap to Click**
  *System Settings → Trackpad → Point & Click*

## Dock
- **Remove Unused Apps**
  *Right-click app → Options → Remove from Dock*

## Battery
- **Show Percentage**
  *System Settings → Control Center → Battery → Show Percentage*
```
